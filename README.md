# Data205__Spring__2026
This repository hosts work related to my capstone class. 

## Project overview and goals

The project hosted on this repository intends to analyze the relationship between crime patterns and income across Montgomery County. This project investigates the assumption that higher income naturally leads to safer environments. 

## Data sources 
The project uses publicly available datasets including: 

**Data montgomery crime report**: For offenses reported to the local police department. 

**US Census Bureau data:** For demographic and income data

## Methodology

**Data collection:**

- Gather data from website in csv format
-  merge datasets based on Police district and year

**Data preprocessing:** 
- Handle missing values
- drop duplicate values
- reformat data,
- convert data types

**Exploratory data analysis:**
- Visualize the evolution of income, population, and crime
- Visualize their distribution
- 

**Statistical modeling:** Use spearman correlation and linear regression to measure the strength of relationships

**Technology used:** 

- Python
- Pandas & Numpy: For data manipulation
- Matplotlib, Seaborn & Tableau: For data visualization
- Statsmodel & Scipy : For statistical analysis
- Jupyter notebook
